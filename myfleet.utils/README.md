# About Code #

- This is the reusable jar including all the exception utilities.

## Tech Stack 

- Java 8
- Springboot

## Setup Guide 

1. Clone the Code : `` git clone git@bitbucket.org:thisara_udaya/myfleet.utils.git -b expnval ``

2. Install Dependencies : `` mvnw install ``
